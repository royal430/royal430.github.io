## 如何使用 Refined Marp Theme

#### 功能介绍

Refined Marp Theme 是一个功能丰富的 Marp 自定义主题，专为演示文稿设计，支持多种页面类型和区块样式。

支持多种主题配色，并提供浅色模式和深色模式

- 浅色模式采用白色背景配合主题色，适用于明亮环境和日间演示场景。
- 深色模式采用深灰背景配合主题色，适用于暗光环境和夜间演示。

#### 使用指南

1. 在 Markdown 文件头部添加 YAML 配置
2. 使用 H2 作为章节标题，H3 作为内容页标题，配合 Marp 的 headingDivider: [2, 3] 配置自动分页，无需手动分页
3. 使用 `<!-- _class: -->` 指定页面样式
4. 使用区块类名创建特殊内容布局
5. 选择合适的主题，例如 rmt-light-blue 或 rmt-dark-blue

**祝您使用愉快！**
